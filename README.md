# Coffee Shop Management System

A web-based Coffee Shop Management System developed as an academic project for managing daily operations within a coffee shop. The system supports sales management, employee management, attendance tracking, inventory management, and promotional activities.

## Project Overview

This project was developed as part of a university software development course. The system aims to improve operational efficiency by centralizing business processes into a single platform.

## Installation

### Backend

```bash
cd backend-master
npm install
npm run start:dev
```

Backend will run on:

```text
http://localhost:3000
```

### Frontend

```bash
cd frontend-master
npm install
npm run dev
```

Frontend will run on:

```text
http://localhost:9000
```
