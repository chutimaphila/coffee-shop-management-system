# ระบบบริหารจัดการร้านกาแฟ (Coffee Shop Management System)

ระบบบริหารจัดการร้านกาแฟที่พัฒนาขึ้นเพื่อช่วยจัดการกระบวนการทำงานภายในร้านแบบครบวงจร ครอบคลุมการจัดการสินค้า การขาย การจัดการพนักงาน การบันทึกเวลาเข้า-ออกงาน การจัดการสต็อก และข้อมูลลูกค้า

## รายละเอียดโครงการ

โครงการนี้พัฒนาขึ้นในรายวิชาพัฒนาซอฟต์แวร์ระดับมหาวิทยาลัย โดยมีวัตถุประสงค์เพื่อออกแบบและพัฒนาระบบสารสนเทศที่ช่วยเพิ่มประสิทธิภาพในการบริหารจัดการร้านกาแฟ และรวบรวมข้อมูลการดำเนินงานไว้ในระบบเดียว

## บทบาทและความรับผิดชอบ

### หัวหน้าทีมโครงการ (Team Leader)

* วางแผนการดำเนินงานและกำหนดระยะเวลาของโครงการ
* แบ่งหน้าที่และประสานงานสมาชิกในทีมจำนวน 9 คน
* ติดตามความคืบหน้าของโครงการและบริหารจัดการงานให้เป็นไปตามแผน
* นำเสนอผลงานและประสานงานระหว่างทีมพัฒนาและอาจารย์ผู้สอน

### ผู้ออกแบบและพัฒนาระบบ

* ออกแบบส่วนติดต่อผู้ใช้งาน (User Interface)
* พัฒนาฟังก์ชันบันทึกเวลาเข้า-ออกงานของพนักงาน
* ร่วมพัฒนาและทดสอบระบบร่วมกับสมาชิกในทีม

## ฟังก์ชันการทำงานของระบบ

### ระบบเข้าสู่ระบบและกำหนดสิทธิ์การใช้งาน

* เข้าสู่ระบบ (Login)
* จัดการสิทธิ์ผู้ใช้งานตามบทบาท

### ระบบจัดการการขาย

* บันทึกรายการขาย
* จัดการคำสั่งซื้อ
* แสดงรายละเอียดการขาย

### ระบบจัดการสินค้า

* เพิ่ม แก้ไข และลบข้อมูลสินค้า
* จัดการหมวดหมู่สินค้า
* จัดการโปรโมชั่น

### ระบบจัดการผู้ใช้งาน

* เพิ่ม แก้ไข และลบข้อมูลผู้ใช้งาน
  
### ระบบจัดการพนักงาน

* จัดเก็บข้อมูลพนักงาน
* บันทึกเวลาเข้า-ออกงาน
* จัดการข้อมูลเงินเดือน

### ระบบจัดการคลังสินค้า

* จัดการวัตถุดิบ
* ตรวจสอบสต็อกสินค้า
* ติดตามข้อมูลคงเหลือในคลัง

### ระบบจัดการลูกค้า

* จัดเก็บและบริหารข้อมูลลูกค้า

### ระบบรายงานและแดชบอร์ด

* แสดงข้อมูลสรุปการดำเนินงาน
* แสดงข้อมูลทางธุรกิจในรูปแบบ Dashboard

## วิธีการติดตั้งและใช้งาน

### Backend

```bash
cd Backend
npm install
npm run start:dev
```

ระบบ Backend จะทำงานที่

```text
http://localhost:3000
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

ระบบ Frontend จะทำงานที่

```text
http://localhost:9000
```

## สิ่งที่ได้รับจากโครงการ

* เรียนรู้การพัฒนาระบบแบบ Full Stack
* ฝึกการออกแบบและพัฒนาระบบสารสนเทศจริง
* พัฒนาทักษะการทำงานเป็นทีมและการบริหารโครงการ
* ฝึกการใช้ Git และ Version Control
* เรียนรู้การออกแบบฐานข้อมูลและการเชื่อมต่อ API
* พัฒนาทักษะการแก้ปัญหาและการวิเคราะห์ความต้องการของผู้ใช้งาน
